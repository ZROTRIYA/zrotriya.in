DESCRIPTION :

The Robotics Workshop includes a detailed study structure, delivered & demonstrated live by our experts in the field of Robotics. Hand Gesture control is an easier approach to control robots instead of buttons or switches. Once the student acquires knowledge about its basic operations, they can further develop & implement it according to their requirements. It can be employed for wide range of application including Home Appliance Control, Motor Control, Wheel Chair Control, Surveillance Robots, etc

Topics Covered

• How to use different Embedded software tools
• How to write program for ATmega16
• How to interface LED with AT mega 16
• How to decode Hand Gestures using Accelerometer
• How to add password protection to the designed Robot
